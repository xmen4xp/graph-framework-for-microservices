## Install data model
```
DATAMODEL_DOCKER_REGISTRY=<container-registry-for-datamodel> TAG=<datamodel-tag> make dm.install
```

Example

```
DATAMODEL_DOCKER_REGISTRY=foo.com TAG=test make dm.install
```
